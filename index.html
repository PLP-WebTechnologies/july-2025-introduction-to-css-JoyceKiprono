INTRODUCTION TO CSS

Learn how CSS is used to style web pages.

What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and design the appearance of HTML elements, such as colors, fonts, spacing, and layouts.

How does CSS work?

  1. CSS can be written inside HTML using the style attribute (Inline CSS).
  2. It can be placed inside the style tag in the HTML file (Internal CSS)
  3. It can be saved in a separate file and linked to HTML (External CSS).

Benefits of CSS

CSS makes web pages more attractive and user-friendly. It allows developers to control the layout, design, and appearance of multiple pages at once.
